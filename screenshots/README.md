# Screenshots Directory

This directory contains screenshots referenced in the documentation.

## Scale Factor Feature Screenshots

The following screenshots should be captured and placed in this directory:

1. **conversion-complete.png**: Shows the interface after successful DXF conversion, highlighting where the Scale Factor panel appears
2. **scale-factor-panel.png**: Shows the Scale Factor Adjustment panel with input field, buttons, and help tooltip
3. **scale-factor-input.png**: Shows the scale factor input field with validation messages and help text
4. **scale-factor-preview.png**: Shows the map preview updating in real-time as scale factor is adjusted
5. **apply-scale-factor.png**: Shows the "Apply Scale Factor" button and processing progress
6. **download-options.png**: Shows the download options with both original and scaled file choices

## Screenshot Guidelines

- Use consistent browser and window size
- Capture at high resolution (at least 1920x1080)
- Include relevant UI elements and context
- Use realistic data examples
- Highlight important features with annotations if needed
- Save as PNG format for best quality
